# Global Yield Curve Comparison

A provisional GitHub Pages version of the BondStats yield curve comparison page.

## What this version does

- compares 2-year and 10-year sovereign yields
- shows 10Y–2Y spread
- labels each curve as normal, flat or inverted
- uses a fixed BondStats comparison dataset

## Current dataset

- United States
- Germany
- United Kingdom
- Japan
- China

## Notes

This version intentionally uses fixed comparison values.

It can later be upgraded to a live multi-maturity JSON feed without changing the front-end structure very much.
